    Démarrage

    1. Ouvrez Paramètres et collez votre clé API AllDebrid ou Real-Debrid.
    2. Cliquez Tester puis Enregistrer.
    3. Dans Récupération, collez l’URL d’une page, indiquez l’hébergeur (ex. rapidgator), puis Récupérer.
    4. Cliquez Résoudre pour obtenir les liens de téléchargement directs.

    Récupération

    URL(s) de page(s)
    Une adresse par ligne. Linkora ouvre chaque page et extrait les liens de l’hébergeur choisi.
    Hébergeur
    Mot-clé filtré dans les liens trouvés (ex. rapidgator, 1fichier). Optionnel : + Ajouter un hébergeur pour en cumuler jusqu’à 6 sur la même page. Après résolution, le 1er hébergeur est prioritaire ; si un lien est mort, Linkora bascule sur un miroir valide. Le bloc principal garde un lien par épisode (prêt pour le téléchargement) ; les doublons/miroirs sont listés à part.
    Récupérer
    Extrait les liens sans encore passer par le débrideur. Les résultats apparaissent par page (un bloc = une URL).
    Historique
    Chaque page sauvegardée peut être rouverte plus tard. Le panneau est replié par défaut.

    Profils — à quoi ça sert ?

    Un profil mémorise un combo de réglages pour ne pas tout retaper : hébergeur + débrideur + nombre de parallèles + retries + template de renommage.

    Exemple : « Rapidgator AllDebrid » → sélectionnez-le, le champ Hébergeur et les options se remplissent.

    • Sauver profil — enregistre la config actuelle sous un nom.
    • Liste déroulante — applique un profil existant.
    • Suppr. — efface le profil sélectionné (pas vos clés API).

    Si vous n’avez qu’une seule façon de travailler, vous pouvez ignorer les profils.

    File d’attente

    Pour traiter plusieurs pages automatiquement :

    1. Collez les URLs + hébergeur.
    2. Ajouter URLs → elles apparaissent dans la file.
    3. Lancer → pour chaque page : récupération puis résolution.

    Utile pour enchaîner plusieurs saisons sans cliquer à chaque fois.

    Résolution (débridage)

    Résoudre
    Envoie les liens au débrideur actif pour obtenir une URL directe.
    Re-vérifier morts
    Retente uniquement les liens marqués morts / erreur.
    Stop
    Interrompt la résolution en cours (file ou lot).
    Parallélisme
    Dans Paramètres : nombre de liens résolus en même temps (1–12).
    Retries
    Nombre d’essais si le lien est temporairement mort.
    Multi-clés
    Vous pouvez coller plusieurs clés API (une par ligne). En cas d’erreur d’auth / quota, Linkora essaie la suivante.

    Filtres, sélection & exports

    • Filtres — valides / morts / recherche texte.
    • Cases à cocher — sélectionner certains liens.
    • Copier / JD — copie presse-papiers ou fichier JDownloader (URL | Nom).
    • CSV / HTML / PDF — exports du lot affiché.
    • Épisodes manquants — détecte les trous type S03E07 absents.

    Renommage intelligent

    Onglet dédié : indiquez un dossier local, Linkora propose des noms propres (Plex / Jellyfin / points).

    • Cochez les fichiers à renommer (tout est coché par défaut) — pratique pour laisser les noms déjà propres.
    • Toujours un aperçu avant de renommer pour de vrai.
    • Le template se choisit dans Paramètres.

    Bibliothèque

    Scannez un dossier ou disque (lecture seule) pour lister films, séries et archives (.zip / .rar / .7z, ex. saison complète). Linkora calcule une identité par média : deux fichiers du même épisode (noms différents) partagent la même clé — base des futurs doublons et du diff PC/NAS.

    • Onglet Bibliothèque (entre Renommage et Aide) — redémarrez l’app / Ctrl+F5 si vous ne le voyez pas.
    • Après scan : vue arbre, liste plate, ou Doublons (même identité, noms différents).
    • Filtres Tout / Séries / Films / Archives + recherche.
    • Archives .zip / .rar / .7z détectées (packs saison).
    • À venir : affiches TMDB.

    Paramètres

    Thème
    Linkora (défaut clair), Ambre (sombre bleu/jaune) ou Pro (sombre dense : en-tête compact fixe, panneaux à liseré, tableaux serrés). Couleur d’accent optionnelle dans Paramètres. Le logo Linkora n’est pas modifiable.
    Notifications
    Alerte système quand une résolution / file est terminée.
    Auto-update
    Vérifie les releases GitHub (ou une URL latest.json perso).

    Backup

    Exporter crée un zip de data/ (historique + réglages / clés). Importer le restaure sur un autre PC ou après réinstall.

    ⚠️ Le zip contient vos clés API en clair : gardez-le pour vous, ne le partagez jamais et ne le déposez pas dans un cloud public.

    Mises à jour

    Au démarrage (ou via « Vérifier les mises à jour »), Linkora compare sa version aux tags GitHub. Cliquez Mettre à jour : une fenêtre de progression montre le téléchargement, puis Linkora se ferme et redémarre tout seul.
