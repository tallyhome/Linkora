# Linkora — Roadmap

Vision produit en **phases**. Chaque phase s’appuie sur la précédente.

**État :** 6 phases au total (0 → 5). **Toutes faites** (phase 5 = nice to have livré).

---

## Phase 0 — Fondations ✅

Socle : extraction, débridage, historique, exports, renommage, thèmes, auto-update, README.

---

## Phase 1 — Distribution & mises à jour ✅

Packaging `.exe`, MAJ zip, `latest.json`, signature (scripts), installateur Inno Setup.

---

## Phase 2 — Qualité des packs & renommage ✅

Épisodes manquants, templates, aperçu avant/après.

---

## Phase 3 — Sélection, filtres & téléchargement ✅

Filtres, sélection multiple, envoi JDownloader.

---

## Phase 4 — Productivité ✅

Profils, notifications, backup `data/`, file d’attente.

---

## Phase 5 — Avancé ✅

| # | Fonctionnalité | Description |
|---|---|---|
| 5.1 | **Multi-clés / rotation** | Plusieurs clés API, bascule si auth/quota |
| 5.2 | **Thème / logo perso** | Accent couleur + logo local |
| 5.3 | **Mode CLI / headless** | `python cli.py …` |
| 5.4 | **Site de MAJ dédié** | Doc `latest.json` hors GitHub |

**Bonus :** page **Aide** (`?` circulaire) — guide des options.
